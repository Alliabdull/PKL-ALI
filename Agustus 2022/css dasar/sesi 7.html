CSS SELECTOR

CSS selector adalah salah satu rule set dari Css yang fungsinya tidak berbeda jauh dengan namanya (Selector) yakni memilih suatu elemen yang ingin kamu beri gaya atau style css.

Id

  • Sebuah elemen HTML hanya boleh memilik 1 id
  • Setiap halaman hanya boleh memiliki 1 elemen dengan id tersebut
  • Dapat Digunakan sebagai penanda halaman untuk link
  • Dapat digunakan untuk javascript
  • Sebaiknya tidak digunakan untuk css(lebih baik gunakan class)
