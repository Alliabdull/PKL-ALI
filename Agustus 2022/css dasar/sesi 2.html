Anatomi Css dasar

seperti ini:

selector {property: value;}

h1 {color : blue ;}

  • Selector adalah sebuah bagian bagian dari CSS yang digunakan untuk memilih dan memanipulasi elemen spesifik pada HTML.
  • Property adalah jenis style atau element yang akan di ubah dalam sebuah tag HTML.
  • Value adalah nilai dari Property.

    • Selector:

  • SELECTOR Digunakan untuk memilih dan memanipulasi elemen spesifik pada html
  • Elemen html dipilih berdasarkan tag ID,CLASS bahkan pola/pattern

    • Property:

  • PROPERTY css dan nilainya itu berguna untuk memberikan efek khusus pada elemen html yang nantinya akan ditampilkan dihalaman web. Anda bisa membuat tampilan dari halaman web lebih menarik dengan berbagai macam properti css.

    • Value:

  • VALUE CSS adalah nilai dari property. Misalkan untuk property background-color yang digunakan untuk mengubah warna latar belakang dari sebuah selector, value atau nilainya dapat berupa red, blue, black, atau white.
