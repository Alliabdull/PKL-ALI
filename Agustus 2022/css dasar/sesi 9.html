CSS inheritance

Definisinya adalah Sebuah Elemen mewarisi beberapa nilai dari properti yang dimiliki oleh elemen parent-nya.

Properti yang diwariskan

  • color
  • font
  • letter-spacing
  • line-height
  • list-style
  • text-align
  • text-indent
  • text-transfrom
  • visibility
  • whute-space

Inheritance dalam CSS adalah sifat penurunan efek CSS dari sebuah tag HTML kepada tag HTML lainnya. Syarat untuk inheritance adalah: tag tersebut harus berada di dalam tag lainnya. Di dalam HTML, setiap tag umumnya akan berada di dalam tag lain.
