Pendahuluan CSS DASAR

CSS adalah Mekanisme sederhana yang mengatur gaya/style pada halaman web CSS sendiri adalah singkatan dari cascading style Sheet

cascading style sheet merupakan aturan untuk mengatur beberapa komponen dalam sebuah web, sehingga akan lebih terstruktur dan seragam. CSS bukan merupakan bahasa pemrograman. CSS bekerja saat sebuah browser dinyalakan agar semua tidak apa.

    Cascading style sheet
  • Aturan yang digunakan untuk menampilkan elemen/tag html.
  • Dibuat terpisah dengan html.
  • Bertujuan untuk memisahkan konten dan style.
  • 1 CSS dapat digunakan dibanyak halaman.
  • 1 halaman html dapat terlihat berbeda jika menggunakan css yang berbeda pula.
