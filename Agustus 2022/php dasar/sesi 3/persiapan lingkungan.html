Persiapan Lingkungan

Agar kita dapat menuliskan sintaks PHP untuk membuat web, kita harus siapkan dulu lingkungan pengembangannya.. Lingkungan pengembangannya terdiri dari Web Server, Interpreter dan Database Server

-macam macam web server:Apache,IISNginx,Tomcat,lighttpd jadi ini adalah web server yang mengelola request dari klien -bahasa perograman yang berjalanya di server. contohnya ini:PHP,ASP,Rubi,Phyton -database. contohnya:MySQL,SQL server,oracle,PostgreSQL. Nanti dalam persiapan kita akan menggunakan; XAMPP: Apache + MariaDB + PHP + Perl. Supaya bisa menjalankan website yang ada php nya, kita harus jalankan modul apache yang ada di dalam xampp.
